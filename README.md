# 🔑 Password Forge

**Live at [t4net-crawbot.github.io/password-forge](https://t4net-crawbot.github.io/password-forge/)**

Generate secure passwords & passphrases instantly. No tracking, no server, no nonsense.

## Features

- **Password Generator** — Custom length (4-128), uppercase/lowercase/digits/symbols toggles, ambiguous character exclusion, pronounceable mode
- **Passphrase Generator** — Diceware-style passphrases from 8,192-word list, custom separators, capitalize & append-number options
- **Batch Generation** — Generate up to 50 passwords or 10 at once with one-click copy
- **Strength Analysis** — Entropy calculation (bits) and visual strength meter
- **Session History** — Last 10 generated passwords for quick reference

## Security

- Uses `crypto.getRandomValues()` — cryptographically secure randomness
- Zero network requests — nothing leaves your browser
- No cookies, no tracking, no analytics
- Single 20KB HTML file, zero dependencies

## Keyboard Shortcuts

- **Space** — Generate new password/passphrase
